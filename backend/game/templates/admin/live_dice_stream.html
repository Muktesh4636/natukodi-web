📱 How to stream from your phone
  1. Download Larix Broadcaster (iOS / Android) — free
  2. Go to Settings → Connections → New connection
  3. Set URL to: rtmp://svs.fightening.sbs/live/<your_stream_key>
  4. Leave Name/Username blank, tap Save
  5. Press the red button to start streaming
  6. After ~5 seconds the status here will show 🔴 LIVE
  7. Users watching /api/game/live-dice/info/ will get the HLS URL automatically
