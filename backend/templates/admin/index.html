What is this page?

This is the database admin for Gundu Ata. Here you can view and edit the raw data stored in the app (users, wallets, bets, rounds, etc.).

  • Core — Users, Wallets, Deposit requests, Payment methods, Transactions.
  • Game — Game rounds, Bets, Dice results, and probability/sound settings.
  • Add — Create a new record. Change — View or edit existing records.

For the game dashboard (deposits, withdrawals, reports), use Game Admin instead.
